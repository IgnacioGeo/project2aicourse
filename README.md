# Project II – Fully Connected Neural Networks (CIIC 5015 – Introduction to AI)

## Overview

This project implements fully connected neural networks using PyTorch 2.0 to solve two tasks:

1. **Regression:** Predicting car fuel efficiency (MPG) using the Auto MPG dataset.

2. **Classification:** Classifying handwritten digits from the MNIST dataset.

The goal is to understand the design, implementation, and training of deep learning models for regression and classification, and to compare different network architectures.

---

## Repository Contents

- `classification_mnist.ipynb`:   
  - Networks with 4 and 6 layers.  
  - Trains each model for 20 epochs with learning rate 0.01.  
  - Report validation loss, accuracy, and training time.  
  - Includes plots and answers to assignment questions.

- `regression_auto_mpg.ipynb`:  
  - Networks with 1, 4, and 5 layers.  
  - Trains each model for 20 epochs with learning rate 0.01.  
  - Reports validation loss and training time.  
  - Includes plots and answers to assignment questions.
